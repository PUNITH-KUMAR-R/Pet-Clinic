# Pet Clinic Management System

## Description

A simple Pet Clinic Management System built with FastAPI, SQLite, SQLAlchemy, and Jinja2.

## Features

- Doctor Management (CRUD)
- Pet Management (CRUD)
- Appointment Management (CRUD)
- HTML UI
- SQLite Database
- Unit Tests
- Security Scanning

## Technologies

- Python 3.11
- FastAPI
- SQLAlchemy
- SQLite
- Jinja2
- Pytest
- Bandit
- pip-audit
- detect-secrets

## Run locally

pip install -r requirements.txt

uvicorn app.main:app --reload
